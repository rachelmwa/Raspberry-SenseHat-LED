# Raspberry-SenseHat-LED

## Description
- Developed python script using the PuTTY system to print and display an output onto Sense hat LED.
- Utilized PuTTY system to code the command for raspberry pi.
- Experienced using hardware and GPIO system.

## How Program Works
- Sensehat LED connected to RPi GPIO
- Python3 command to program a word to display on SenseHat LED
- Program executed via PuTTY 

## Hardware Used
- Raspberry Pi 4
- SenseHat

## Software Tools
- PuTTY IDE (SSH)
- Python
- RPi OS

## What I Gained
- Using GPIO and a hardware embedded system
- Python programming
- Remote development
- Software and hardware debugging
- Microcontroller experience
